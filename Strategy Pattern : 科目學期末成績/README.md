# Problem Description
撰寫一個程式供使用者輸入以下科目的成績，
依序輸入該科 出席、作業、期中、期末 成績

(需使用strategy pattern)
並依照該科目算分標準 計算出結果：

- 國文 平時 ( 出席30% + 作業70% ) 40%, 期中 20% , 期末 40%
- 英文 平時 ( 出席40% + 作業60% ) 40%, 期中 30% , 期末 30%
- 組合語言 平時 ( 出席50% + 作業50% ) 0%, 期中 0% , 期末 100%
